# More Sections Make a Better Learning Card

## 1. 错误说法
- If a learning card has more sections, it must be more complete and therefore better.

## 2. 为什么它看起来像对的？
- A fuller template looks more rigorous, so people confuse visible completeness with actual retrieval value.

## 3. 它错在哪里？
- Extra sections often repeat nearby ideas instead of adding new signal.
- More visible scaffolding can hide the actual answer the card is supposed to preserve.

## 4. 正确理解
- A better learning card keeps the minimum set of sections that preserve answer quality, graph placement, and next action.

## 5. 它混淆了什么？
- It confuses template completion with knowledge compression.

## 6. 代表性反例
- A short card with one clean definition, one boundary rule, and two strong graph links can outperform a bloated card with five weak reflection sections.

## 7. 触发信号
- The reviewer hesitates to delete a section only because the template provides it.
- Most new content lands in status, checklist, or reflection blocks rather than in the main answer.

## 8. 纠偏动作
- Score each section by retrieval value and remove or gate sections that do not change understanding, routing, or action.
